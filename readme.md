# Tutorial jQuery: Actividad 2

Este proyecto consiste en una guía práctica y tutorial sobre la implementación de la biblioteca **jQuery** para el desarrollo de interfaces web dinámicas.

## Descripción del Proyecto

La actividad se enfoca en el uso de **AJAX** para realizar consultas asíncronas a una API externa (JSONPlaceholder), permitiendo la actualización de datos en el DOM sin necesidad de recargar la página.

## Tecnologías Utilizadas

* **HTML5 / CSS3**: Estructura y diseño de la interfaz.
* **JavaScript**: Lógica de programación.
* **jQuery**: Biblioteca utilizada para la manipulación del DOM y gestión de peticiones AJAX.
* **GitHub Pages**: Plataforma utilizada para el despliegue del sitio web.
