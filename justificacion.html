Justificación del Framework

jQuery es una biblioteca de JavaScript diseñada para simplificar la interacción entre HTML y JavaScript[cite: 1].

¿Para qué sirve?

Facilita la selección de elementos DOM, el manejo de eventos, la creación de animaciones y la realización de peticiones asíncronas (AJAX) de manera más sencilla que con JavaScript nativo[cite: 1].

Aplicaciones y prerrequisitos

Se utiliza en aplicaciones web que requieren dinamismo sin recargar la página. Se apoya en lenguajes como HTML, CSS y JavaScript, y como prerrequisito principal se requiere tener conocimientos básicos de la sintaxis de JavaScript[cite: 1].
