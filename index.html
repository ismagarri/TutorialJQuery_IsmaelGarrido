Tutorial de jQuery

Actividad 2: Desarrollo de actividades en red

Autor: Ismael Garrido Collantes

Bienvenido a esta guía práctica. El objetivo de este tutorial es sintetizar el conocimiento necesario para aplicar jQuery en la resolución de problemas web, facilitando la manipulación del DOM y la comunicación asíncrona.

Navega a través del menú superior para explorar la justificación técnica, los pasos de instalación y una aplicación práctica utilizando AJAX.
